# ⏰ Analog Clock (Python GUI)

An aesthetic and animated analog clock built using Python's Tkinter library.  
This project displays a real-time clock with smoothly moving hour, minute, and second hands along with a clean UI design.

---

##  Features
-  Real-time analog clock
- Accurate hour, minute, and second hands
-  Clock numbers (1–12) displayed
-  Clean and aesthetic dark UI
-  Smooth animation using Tkinter

---

## Technologies Used
- Python 
- Tkinter (GUI Library)
- Math & Time modules
